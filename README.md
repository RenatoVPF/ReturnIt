## Return it ,hero!

É um jogo feito como aprte do processo seletivo para entrada na liga acadeimica de jogos do DCOMP/UFS: WADS

Sobre: 
“Return it, hero!” é um jogo de navegação e resolução de problemas no qual o herói, após derrotar o rei demônio, deve devolver as habilidades que pegou emprestado durante a jornada, planejando
sua rota de acordo com quais habilidades precisa para alcançar a próxima localização.

Implementação:
* Engine: Unity
* Música: BeepBox
* Efeitos Sonoros: Sjfxr
* PixelArt/Interface: LibreSprite
